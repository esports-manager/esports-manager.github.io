Project Roadmap

This page outlines the planned milestones, current progress, and how you can get involved with eSports Manager.

Last updated: 2025-08-12

Milestones

v0.1 — Prototype (Playable loop)

  • Core data model (teams, players, matches)
  • New web-based UI
  • Basic CRUD APIs
  • Player and champion generation
  • Default database
  • New match simulation engine (text-based)
  • Save/Load system

v0.2 — Content & UX

  • Expanded database and editors
  • Tutorial/onboarding
  • Improved accessibility and theming
  • Performance optimizations

Upcoming

Short term (next 1–2 months)

  • Public playtest build
  • Bug bash and telemetry
  • Polish text commentary

Medium term

  • Additional esports modes (RTS/FPS-inspired)
  • Modding improvements
  • PWA packaging

About this roadmap

This roadmap is a living document and may change based on community feedback, feasibility, and priorities. If something feels missing or misprioritized, please open an issue.
